BitByBit

Visual Programming of Geometry

BitByBit is an open source frictionless web platform that offers visual development environment for programming geometry. This tool is great for all ages and can fulfil many artistic, educational, and practical needs of its users. Through familiar programming interface used in tools such as Scratch we expose powerful 3D algorithms that make it easier to implement various parametric tasks.

school Learn

BitByBit is focused on education. The platform is built in a way that it would be simple to use for beginners. Visual programming language abstracts away many complexities of writing code in a traditional way. If you are starting out our YouTube channel has videos that will walk you through some of the basics of programming and writing scripts in BitByBit.

edit Create

When you become familiar with BitByBit you will see that this tool can be used to solve practical everyday 3D tasks. You can create scripts that perform various modeling, data visualisation, manufacturing, analysis or other workflows. We want to provide you with the best building blocks that could be practical and helpful for you.
